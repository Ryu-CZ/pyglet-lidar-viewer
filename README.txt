Lidar Data Viewer

Simple viwer of Lidar point cloud data. Visualize point in 3D openGL. 


-------
Setting
-------
Set file_path variable in script lidarViewer.py. (Path to bin file of lidar point cloud.)


---------
Controls
---------
Rotate camera:
    -left mouse button + move mouse
Zooming:
    -scroll(affects speed of move)
Move:
    -horizontal: arrows or "WASD"
    -vertical: pg_up, pg_dn
    -reset position: "R"
